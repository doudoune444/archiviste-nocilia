# Plan — ING-001 Ingestion `lore/*.md` (parse + chunk + embed + UPSERT idempotent)

## 1. Résumé

Pipeline CLI Python one-shot `python -m archiviste_workers.ingest --path lore/` qui parcourt les `.md`, parse le frontmatter YAML, normalise NFKC, chunke via `RecursiveCharacterTextSplitter` (tokenizer bge-m3, 512/64), embedde via `BAAI/bge-m3` (sentence-transformers), et UPSERT idempotent dans `documents` + `chunks` (asyncpg, transaction par fichier) avec hash SHA-256 du corps normalisé.

## 2. Hypothèses résolues

- **DB driver** : `asyncpg` direct (déjà dep), pas SQLAlchemy. Codec pgvector via `pgvector.asyncpg.register_vector(conn)` au pool init. Justification : decision spec line 107.
- **Tokenizer chunker** : `transformers.AutoTokenizer.from_pretrained('BAAI/bge-m3')` passé à `RecursiveCharacterTextSplitter.from_huggingface_tokenizer(...)`. Ajoute `transformers` (déjà tiré transitif par sentence-transformers, pas de nouvelle dep top-level).
- **Frontmatter parser** : `yaml.safe_load` sur le bloc entre les deux `---` (regex `^---\n(.*?)\n---\n(.*)$` avec DOTALL). `pyyaml` ajouté à pyproject. Pas de dep `python-frontmatter` (1 fonction triviale, YAGNI).
- **CLI framework** : `argparse` stdlib (1 flag `--path`, pas de sous-commandes). Pas de `click`/`typer`.
- **Repo root detection** : remontée filesystem depuis `Path.cwd()` jusqu'à dossier contenant `.git/`. Path résolu via `Path.resolve(strict=True)`, vérifié `is_relative_to(repo_root)`.
- **Logger** : `structlog` avec `JSONRenderer`, configuré dans `archiviste_workers.logging_config` (nouveau module partagé minuscule). Output stdout.
- **Embedder lifecycle** : singleton chargé à l'init du process CLI (pas de FastAPI). Téléchargement HF cache standard `~/.cache/huggingface/`. `SentenceTransformer('BAAI/bge-m3', cache_folder=None)` — laisse HF gérer.
- **Pool DB** : `asyncpg.create_pool(min_size=1, max_size=2)` — séquentiel, 2 connexions max suffisent.
- **Order chunks dans splitter** : `RecursiveCharacterTextSplitter.split_text` retourne liste ordonnée → `enumerate` donne `ord` 0-indexé direct.
- **Tags array** : passé à asyncpg comme `list[str]` Python natif → cast PG `text[]` automatique.
- **Test DB** : fixture pytest qui démarre/arrête une DB jetable via `docker compose` (postgres existant) + `make migrate` ; isolation via schémas PG ou DROP/CREATE entre tests (choix simple : transaction-per-test rollback).
- **Property INV-1** : tolérance overlap ≥ 0 ; assert `sum(len(c) for c in chunks) >= len(text)` et `sum <= len(text) + (n-1) * chunk_overlap` (overlap-modulo).
- **Property INV-2** : tirer N strings non-vides via hypothesis, embedder.encode(batch).shape == (N, 1024).
- **Network mock AC-9** : monkeypatch `huggingface_hub.snapshot_download` à raise après warm-up dans le test, encode appelé une 2ème fois → succès car cache hit.
- **Sample lore** : 2 fichiers `.md` minimaux écrits manuellement (titre + 3-5 paragraphes lore Nocilia placeholder), zéro PII.
- **Exit code** : `sys.exit(1)` si compteur erreurs > 0 sinon `sys.exit(0)`. Catch propres `try/except` autour de chaque fichier ; échec global init (DB, modèle) → exit 2 distinct.
