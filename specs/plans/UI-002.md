## Test strategy
- **Intégration** `#[sqlx::test]` (Z3 — Postgres CI déjà câblé) seedant 3 tickets `open` (prio 5/3/3 `created_at` distincts) + 1 `resolved` + 1 `dismissed` + 1 `conversations` row `gcs_uri='gs://archiviste-conversations/conv/<id>.md'`. JWT `author`/`member` via helper SEC-001 (`AuthUser` extractor). Mapping : AC-5 (ordre strict + total/limit/offset), AC-6 (member 403 + anonyme 403 byte-for-byte), AC-7 (table de cas `limit=0,-1,201,abc` + `offset=-1,foo` → 400), AC-8 (200 shape), AC-10 (a UUID non existant 404 / b non-UUID 400 / c member 403), AC-11 (assertion router : `/v1/tickets` + `/v1/conversations/{id}/signed-url` portent `AuthUser` extractor, pas `#[public]`), AC-12 (4 headers UI-001 préservés sur les 2 routes), AC-19 (montage `lib.rs`, body limit 1 MiB GEN-002 reste scopée `/v1/chat`), AC-23 (capture tracing subscriber : events nommés + champs autorisés ; assert absence `question` / `signed_url` / `gcs_uri` en clair).
- **Unit** (`test_signed_url.rs`) : génération clé RSA 2048 runtime via `rsa` (dev-dep Z4) → `sign_get("conv/abc.md", now)` ; assert URL prefix `https://storage.googleapis.com/archiviste-conversations/conv/abc.md?`, query contient `X-Goog-Algorithm=GOOG4-RSA-SHA256`, `X-Goog-Expires=300`, `X-Goog-SignedHeaders=host` ; signature hex 512 chars ; `expires_at - now ∈ [298,302]s` ; assertion `SIGNED_URL_TTL_SECONDS == 300` (AC-9, AC-21).
- **Contract** : AC-20 grep `sqlx::query!` + SQL littéral `ORDER BY priority_score DESC, created_at DESC` + `WHERE status='open'`. AC-22 = CI bash step `git diff --exit-code specs/openapi/` (Z5).
